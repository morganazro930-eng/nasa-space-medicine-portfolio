# Wearable Recovery Dashboard Plan

## Dashboard sections

1. Participant/session overview
2. Sleep and recovery trends
3. Physiological stress indicators
4. Mobility or performance outcomes
5. Flags for follow-up or data quality review

## Key design principle

The dashboard should help a researcher or clinician quickly understand whether recovery, sleep, or performance is improving, stable, or worsening.

## Privacy note

Use synthetic data only.
